# GACERC - Sitio Web Retro Gaming

## 📖 Descripción
Este proyecto es una página web informativa multipágina construida a medida, diseñada con una temática de videojuegos retro y *pixel art*. Está enfocada en ofrecer una navegación fluida y una experiencia de usuario inmersiva, sirviendo como catálogo visual de productos y plataforma de contacto interactiva.

## 🚀 Tecnologías Utilizadas
* **HTML5:** Estructura semántica multipágina y maquetación avanzada de formularios.
* **CSS3:** Estilos personalizados, paleta de colores temáticos de alto contraste y animaciones.
* **Bootstrap:** Sistema de grillas para asegurar un diseño completamente adaptable a cualquier dispositivo.
* **JavaScript:** Lógica de interactividad básica y manipulación del DOM.

## ✨ Características Principales
* **Navegación Multipágina:** Secciones enlazadas e independientes (Inicio, Sobre Nosotros, Productos, Contáctanos) para una arquitectura de información clara.
* **Diseño Temático:** Interfaz responsiva con una identidad visual fuertemente inspirada en el *retro-gaming*.
* **Módulo de Contacto:** Formulario diseñado con botones de acción y múltiples etiquetas de ingreso/salida de datos, con la estructura técnica preparada para el procesamiento de información mediante métodos GET y POST.
